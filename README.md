# FIN-ED INDIA
A multilingual, scenario-based financial literacy platform designed to make financial education simple and accessible for everyone.

## Problem Statement
A website app that teaches basic finance topics(banking, savings, loans, taxes, UPI safety)in multiple Indian language .

## Solution
FIN-ED INDIA provides financial education in regional languages using real-life scenarios and interactive quizzes.

## Key Features
- Multilingual support
- Scenario-based learning
- Simple quizzes
